# SimpleLogger
SimpleLogger is very simple logging tool in Swift.

# How to use
```
SimpleLogger.error()

SimpleLogger.warning("message")

SimpleLogger.info("a", "b", "c")

SimpleLogger.debug(".debug is only available in Debug Build in basically")
```
